##### if else 语句
```c
Node.type = NODE_TYPE_DECORATOR;
node.decorator->type = DECORATOR_TYPE_CONDITIONAL;
executeNode(node);
```
效果等价如下
```c
if (node.decorator->condition()) {
    return executeNode(node.children[0]);
}
else {
    return executeNode(node.children[1]);
    return 0;
}
```
